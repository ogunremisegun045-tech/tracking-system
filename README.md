# LiveTrack - Live Order Tracking System

A full-stack PHP web application for real-time order tracking with interactive maps.

## Features
- User registration and login
- Live map tracking with Leaflet.js
- Admin panel to manage orders and users
- Auto-refreshing map every 10 seconds
- Order status updates (Pending, Shipped, Delivered, Cancelled)
- GPS coordinates for accurate location tracking

## Technologies Used
- PHP
- MySQL
- JavaScript
- Tailwind CSS
- Leaflet.js (OpenStreetMap)
- XAMPP

## How It Works
1. Admin creates an order and assigns it to a customer
2. Customer logs in and tracks their order
3. Map shows real-time location of the package
4. Admin updates location as package moves
5. Map auto-refreshes every 10 seconds
